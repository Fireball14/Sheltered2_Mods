This mod adds temperature modifier to all greenhouses:
Small +-5 degrees. 
Large +-10 degrees. 
Huge +-15 degrees. 

NOTE: Mod modifies min/max temperature when plant is planted. This means only plants planted after this mod was enabled will get modified.


### How to use:
1) BACKUP YOUR SAVES!!!
2) Download BepInEx 5.4.19  - https://github.com/BepInEx/BepInEx/releases
3) Navigate to your game folder and extract BepInEx archive there.
4) Now run the game. Exit.
5) Go in to BepInEx\plugins\ folder and put GreenhousePlus.dll there. Last build can be found [here](Release/)
